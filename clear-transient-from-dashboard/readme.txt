If you are logged in as a user who can manage options, you can clear all transients from the dashboard.


== Description ==

If you are logged in as a user who can manage options, you can clear all transients from the dashboard.
When the display does not change, it is convenient to immediately check whether it is due to a transient or a bug.


== Installation ==

1. Upload `clear-transient-from-dashboard` to the `/wp-content/plugins/` directory.
2. Activate `Clear Transient From Dashboard` through the 'Plugins' menu in WordPress.
