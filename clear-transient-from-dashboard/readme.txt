== Description ==

Logging in as admin, you can clear all transients from the dashboard.
If you can find no change in your screen, it is convenient to immediately check whether it is due to a transient or a bug.


== Installation ==

1. Upload `clear-transient-from-dashboard` to the `/wp-content/plugins/` directory.
2. Activate `Clear Transient From Dashboard` through the 'Plugins' menu in WordPress.
